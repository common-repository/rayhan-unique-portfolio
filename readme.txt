== Description ==

Image portfolio with grid view.On hover shows title.Click and see full image with title and description.Different type of views with moving arrows.Use shortcode [rayhan_unique_portfolio].Bottom 2 buttons are for change view of the single image and gallery.User can make the image 100% large and also can change the view of the portfolio easily.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. You will see an option called Unique Portfolio on the left admine section.
4. Add your gallery from here.Put title,Description and featured image.
5. Use the Shortcode [rayhan_unique_portfolio] for show products on page or post.

== Frequently Asked Questions ==

= Where can I find Shortcode for it? =

Use [rayhan_unique_portfolio]

= Can i put shortcode on page or posts? =

Yes you can.But you need to make sure your post/page editor support shortcode.

= Can i change image size? =

Yes you can if you know basic php and css.

= Can i add the gallery everywhere? =

Yes you can.Just use shortcode for it.
